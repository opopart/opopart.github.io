值和单位

发表于 2020-07-17 | 分类于 基础 ， CSS

RGB颜色

​ 整数范围0~255，百分数0%~100%；如果超出可取范围，会自动“剪裁”到最近的范围边界。

Web安全颜色

​ 20%、51、十六进制的33

​ 不同平台有着不同的调色板，不同浏览器也有着自己的调色板，通过设置安全颜色，使其称成为标准

长度单位

​ em: 1个‘em’等于一种给定字体的font-size值

​ rem：font-size值基于根源素设置，大多数情况下为html元素

​ ex：所用字体中小写x的高度

​ vh：viewport高度的1/100

​ vw：viewport宽度的1/100

​ vmin：本次高度和宽度的最大值的1/100

​ vmin：本次高度和宽度的最小值的1/100，例，如果浏览器高度为1100px,宽700px,则1vmax = 11px, 1vmin = 7px;
