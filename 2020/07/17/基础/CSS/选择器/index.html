选择器

发表于 2020-07-17 | 分类于 基础 ， CSS

元素选择器

​ 如：html、body、h1、a、p

类选择器

​ 前面加.

ID选择器

​ 前面加#

属性选择器

一、简单属性选择

​ 如果希望选择有某个属性的元素，而不论属性的值是什么，可以选用简单属性选择

​ 如：h1[class] {} 选择有class属性的所以h1元素

​ *[title] {} 选择所有含有title属性的元素

​ a[href][title] {} 选择含有href和title属性的所有a标签

二、根据具体属性值选择

​ 范围更小，更具体

​ 如：a[href="xxx"] {} 选择选择href链接为xxx的所有a元素

三、根据部分属性值匹配

​ [class~=’waring’] ： 选择class属性中包含waring的元素

​ [class^=’bar’] ：选择class属性以‘bar’开头的所有元素

​ [class$=’bar’] ：选择class属性以’bar’结尾的所有元素

​ [class*=’bar’] ：选择class属性包含字串’bar’的所有元素

四、特定属性选择

​ *[lang|=’en’] {} ：选择lang属性等于en或者en-开头的所有元素

​

后代选择器

​ 用空格隔开

​ 如： h1 em {} ： 表示作为h1元素后代的任何em元素

选择子元素

​ 某些情况下并不想选择一个任意的后代元素，而只选择某个元素的下一代，仅仅指下一层(文档的树结构)

​ 用 > 表示

​ 如 h1 > strong {} 对一个起效果，对第二个不起效果

选择相邻兄弟元素

​ 选择紧接在另一个元素后的元素，而且二者有相同的父元素， 使用 + 号

伪类选择器

​ 可以为文档中不一定具体存在的结构指定样式

​ 链接伪类：

​ 1.:link 任何超链接，包括已访问和为访问的超链接

​ 2.:visted 已访问地址超链接的所有锚

​ 动态伪类：

​ 1.:focus 指示当前拥有输入焦点的元素

​ 2.:hover 鼠标停留时

​ 3.:active 激活时

​ :first-child 选择第一个子元素

​ :nth-child(n) 选择第n个子元素

​ :nth-of-type(n) 选择第n个子元素，注意，是在一堆兄弟元素中选择相同标记元素中的第n个

伪元素选择器 用 ::

​ 伪元素能够在文档中插入假想的元素，从而达到某种效果。

​ :first-letter 首字母样式，只能用于标记或段落之间的块级元素

​ :first-line 首行样式，只能用于标记或段落之间的块级元素

​ :before {content: ""; color: silver} 在某个元素前添加内容，

​ :after {content: ""; color: silver} 在某个元素后面添加内容；
